# Shopping cart

## Инструкция по запуску приложения **локально**

_Приложение использует [Node.js](https://nodejs.org/) для своей работы_

```
$ git clone https://github.com/ildarmjs/shopping_cart.git
$ cd shopping_cart/
$ npm i
$ npm run dev
```
